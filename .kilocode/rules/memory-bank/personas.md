# User Personas

UXcaptain serves B2B companies conducting video and audio user research, with personas tailored to Spain's digital-savvy market emphasizing personal relationships, hybrid digital-physical strategies, and sectors like tech, telecom, e-commerce, and tourism.

## Persona 1: UX Researcher in Tech Agency

**Name:** Marta López  
**Age:** 32  
**Role:** Senior UX Researcher at a Madrid-based agency like Creative Navy or Crater, managing user testing for software and fintech clients.  
**Company:** Mid-sized digital agency (11-50 employees), focused on UX/UI for B2B SaaS.  

**Goals:** Recruit participants quickly for screen/audio recordings, analyze insights via transcription microservices, ensure GDPR compliance for Spanish users.  

**Pain Points:** Time-consuming manual recruitment; need for cost-effective tools amid €1,000+ project budgets; integrating with React frontends and Express backends.  

**Behaviors:** Uses LinkedIn and Instagram for networking; values all-in-one platforms like UXtweak for Spain-specific panels; attends Barcelona UX events.  

**Quote:** "Necesito una herramienta que maneje reclutamiento local y transcripciones rápidas para iterar prototipos."
